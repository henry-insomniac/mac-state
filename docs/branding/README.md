# Branding

This directory contains the current `mac-state` logo assets.

- `mac-state-logo.svg`
  Source vector for the logo concept.
- `mac-state-logo-1024.png`
  Master raster export used to generate app icon sizes.
- `mac-state-logo-preview.png`
  Quick preview board for dark and light backgrounds.
- `mac-state.icns`
  Native macOS icon bundle export.

The mark combines a menu bar strip with live metric bars and a single active status indicator to reflect the app's focus on lightweight menu bar monitoring.
